# Knowledge: Infrastructure Stack

| Tech | Best Use Case | 2026 Context |
| :--- | :--- | :--- |
| **PostgreSQL** | Relational, ACID compliance. | Default for structured data. |
| **MongoDB** | Document-based, flexible schema. | High-volume JSON/Logs. |
| **Redis / Valkey** | In-memory cache/store. | Use **Valkey 8+** for fully open-source BSD threading. |
| **SQL Server** | Enterprise legacy/integration. | Strict T-SQL standards. |
| **Kubernetes** | Orchestration. | Use Helm charts for deployments. |
