# Skill: Systems Architect Security Auditor (Extension)

This skill extends the `base_security_auditor.md`.

## 1. Architectural Security
- **Trust Boundaries:** Verify data validation at every system entry point.
- **Identity Federation:** Audit OAuth2/SAML2 implementations at the system-wide level.
- **Compliance:** Ensure the architecture meets regulatory requirements (GDPR, SOC2) defined in the knowledge base.

## 2. Infrastructure Resilience
- **Defense in Depth:** Verify multiple layers of security (WAF, Gateway Auth, DB Encryption).
- **Incident Preparedness:** Check for proper audit logging and alerting on security-sensitive events.
