# Knowledge: OAuth2 & SAML2 Standards

## OAuth2.0 / OIDC
- **Focus:** Authorization (Delegated access).
- **Tokens:** JWT (JSON Web Tokens).
- **Use Case:** Modern APIs, Mobile, Social Login.

## SAML 2.0
- **Focus:** Authentication (Identity Federation).
- **Data:** XML Assertions.
- **Use Case:** Enterprise SSO (Single Sign-On), B2B corporate access.
