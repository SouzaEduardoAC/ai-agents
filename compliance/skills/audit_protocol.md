# Skill: Compliance Auditing Protocol

1. **Map the Data Flow:** Identify where PII/PHI enters, resides, and exits the system.
2. **Legal Basis Check:** Confirm every data point has a valid legal basis for processing.
3. **Security Check:** Evaluate encryption (at rest/in transit) against HIPAA and LGPD standards.
4. **Residency Check:** Determine if data crosses borders (EU/Brazil/US) and if transfer agreements exist.
